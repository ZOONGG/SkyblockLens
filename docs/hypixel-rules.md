# Hypixel Rules Summary For SkyBlock Lens

SkyBlock Lens only implements client-side quality-of-life features that help
display, search, organize, or explain information. It must not automate
gameplay or bypass server rules.

## Allowed Direction

- HUD information from client-visible scoreboard, tablist, actionbar, chat, or
  local data.
- Tooltips, item browser, local recipe information, settings, filters, timers,
  and visual overlays.
- Explicit, transparent, opt-in API requests for future profile or market data.

## Disallowed Direction

- Automatic clicking, farming, mining, fishing, combat, movement, or terminal
  completion.
- Sending automatic commands unless the user explicitly clicks a clear and safe
  UI action.
- Packet manipulation, ghost blocks, anti-knockback, reach, aim assist, or
  anti-cheat bypass.
- Hidden network behavior or data collection.

When a feature is ambiguous, default to not implementing it until it can be
shown to be a purely visual or informational QoL feature.
